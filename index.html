  • tag parts of speech in each sentence,

  • insert into global collection.

parser_package.update_database_from_excel_review module¶

update_database_from_excel_review.py¶

This module contains a set of functions that read the reviewed learning objectives from the excel file and update the database. If a new learning objective has been added to the excel file, it will be added against the appropriate course. If a learning objective has been designated as REMOVE then it will be removed from the database.
